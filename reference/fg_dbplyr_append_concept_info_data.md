# fg_dbplyr_append_concept_info_data

fg_dbplyr_append_concept_info_data

## Usage

``` r
fg_dbplyr_append_concept_info_data(dbplyr_table, fg_bq_tables, ...)
```

## Arguments

- dbplyr_table:

  a tbl object with the data

- fg_bq_tables:

  an object of type fg_bq_tables with the bigquery tables

- ...:

  see `fg_append_concept_info_data_sql` for the mapping options

## Value

tbl with added columns
